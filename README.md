# 🚀 Modern Personal Portfolio - Chamodi Imasha

Welcome to the repository of my professional personal portfolio! This is a modern, responsive, and high-performance website designed to showcase my skills, projects, and professional background.

## ✨ Features

- **💎 Modern UI/UX**: Premium dark theme with glassmorphism effects and vibrant gradients.
- **📱 Fully Responsive**: Optimized for all devices—mobile, tablet, and desktop.
- **🎬 Dynamic Animations**: Smooth scroll-reveal effects, custom cursor, and interactive particles.
- **🛠️ Project Showcases**: Beautifully designed project cards with direct links to GitHub and Figma.
- **📄 CV Integration**: Integrated PDF viewer and download options for my professional resume.
- **📩 Real-time Contact Form**: Fully functional contact form powered by Formspree.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure for better SEO and accessibility.
- **CSS3**: Custom properties (Variables), Flexbox, Grid, and Keyframe animations.
- **JavaScript (Vanilla)**: Interaction logic, intersection observers, and UI enhancements.
- **FontAwesome**: High-quality vector icons.
- **Google Fonts**: Modern typography (Space Grotesk & Inter).

## 📂 Project Structure

```bash
├── index.html          # Main portfolio page
├── style.css           # Custom CSS animations and styles
├── script.js           # Interactive UI logic
├── images/             # Profile photos and asset images
└── Chamodi Imasha Sewwandi CV.pdf  # Professional CV
```
